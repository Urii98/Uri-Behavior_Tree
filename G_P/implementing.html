Implementing Behavior Trees

The best way to learn how to use a new technology or library is by putting it into practice. In this section, you will have the opportunity to apply the concepts and techniques learned in the previous sections by implementing two behavior trees using the library I have created. The exercise is designed to reinforce your understanding of how behavior trees work and how to use my Behavior Tree implementation to create complex AI behaviors. Remember to follow the conventions for using the behavior tree class in order to be successful. I encourage you to try the exercise on your own before checking the solution to get the most out of this learning experience.

Project

You can start the exercise by downloading the project from the GitHub repository provided. To make it easier for you to get started, I will provide the initial part of the exercise for both exercises: the diagram of the behavior tree. However, it is up to you to implement the rest of the exercise by following the conventions for using the behavior tree class. This will help you gain a better understanding of how to use the library and how behavior trees work. So, take your time, work through the exercise at your own pace, and feel free to refer to the documentation and examples provided as needed. Good luck!
